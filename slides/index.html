08 · why we structured it this way

Three layers, on purpose

domain brains — one per teammate, they own judgment
Entry skills (the only thing the prompt names)
$scout $fashion-designer $builder $performance-marketer
parallel specialists — do one job, hand back results
Subagents
x-researcher exa-researcher cap / sock / apparel-designer fashion-reviewer drop-site-builder · reviewer · deployer ad-copywriter · ad-operator
domain-blind tools — they don't know fashion exists
Reusable skills
$x-trends $exa-search $imagegen $frontend-skill $agent-browser $meta-ads-cli
e.g. $imagegen serves the Fashion Designer's mockups and the Builder's site imagery — build a tool once, every teammate gets it
