# References

Project material that isn't shipping app code — documentation, design assets, and read-only external checkouts.

- `docs/` - All Markdown documentation. Start at [`docs/README.md`](docs/README.md).
- `docs/specs/` - One spec per teammate, numbered in pipeline order.
- `whiteboard/` - Architecture diagrams and the original whiteboard photos.
- `uimocks/` - Product and UI mockups, screenshots, and demo GIFs.
- `slides/` - The "how it works" presentation deck, served at `/slides`.
- `codex-sdk/` - Read-only checkout of the official OpenAI Codex TypeScript SDK (upstream `openai/codex`), kept for API inspection only. Do not edit.
- `sandbox-prototypes/` - The early Phase A tutorial prototype that proved the execution substrate: Vercel Sandbox snapshots running the Codex SDK.
